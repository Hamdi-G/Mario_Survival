# Mario survival
Jeu mulitijoueurs inspiré du monde de Super Mario

## PRESENTATION
Prototype de jeu qui peut se jouer à plusieurs avec un chat virtuel mis à disposition des joueurs.

## But du jeu
Mario doit récupérer les coins en evitant les ennemis
